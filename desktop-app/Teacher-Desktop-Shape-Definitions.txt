TEACHER DESKTOP FLOWCHART - SHAPE DEFINITIONS
Based on Admin-Desktop.drawio patterns
==========================================

SHAPE TYPE MAPPING:
- Start/End: ellipse (oval)
- Process: rounded=0 (rectangle)
- Decision: rhombus (diamond)
- Input/Output: shape=parallelogram (parallelogram)
- Connector: shape=offPageConnector (off-page connector)
- Merge Point: shape=orEllipse (circle for merging flows)
- Section Title: text with fontSize=20, fontStyle=3

==========================================
SUMMARY OF SHAPE PATTERNS
==========================================

ELLIPSE (Start/End):
- Start node
- STD connector (when used as endpoint)

RECTANGLE (Process):
- All "Load X View" operations
- All "Load X" data operations
- All "Update X" operations
- All "Create X" operations
- All "Delete X" operations
- All "Validate X" operations
- All "Mark X" operations
- All "Reset X" operations
- All "Call X" operations
- All "Processing X" operations
- All "Apply X" operations
- All "Reload X" operations
- All "Save X" operations
- All "Filter X" operations
- All "Query X" operations

RHOMBUS (Decision):
- All questions ending with "?"
- All conditional checks
- All "X?" patterns

PARALLELOGRAM (Input/Output):
- All "Show X" operations (screens, views, modals, dialogs)
- All "Display X" operations
- All "Input X" operations
- All "Select X" operations
- All "Modify X" operations
- All success/error messages
- All "View X" operations

OFF-PAGE CONNECTOR:
- All feature connectors (VSP, VGE, VSM, VMA, VMS, PRO, SET, NOT, HEL)
- STD connector (for returning to dashboard)

OR-ELLIPSE (Merge Point):
- Points where multiple flows converge before returning to dashboard
