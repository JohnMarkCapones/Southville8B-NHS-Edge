# Quick Start: Distribute Your App in 5 Minutes

## 🚀 Fastest Method: Google Drive

### Step 1: Download APK
1. Go to Expo dashboard: https://expo.dev/accounts/southville8bnhs/projects/Southville-Mobile/builds
2. Find your latest successful build
3. Click "Download" to get the APK file

### Step 2: Upload to Google Drive
1. Go to drive.google.com
2. Create folder: "Southville Mobile App"
3. Upload the APK file
4. Right-click → "Share" → "Anyone with the link" → Copy link

### Step 3: Share with Users
Send this message to users:

```
📱 Southville Mobile App is Ready!

Download: [PASTE_GOOGLE_DRIVE_LINK]

Installation:
1. Enable "Install from Unknown Sources" in Settings → Security
2. Tap the link above and download
3. Tap the APK file when download completes
4. Tap "Install anyway" if warned (this is safe!)
5. Open the app

Full guide: [LINK_TO_INSTALLATION_GUIDE]

Need help? Contact: [YOUR_SUPPORT_EMAIL]
```

**That's it!** Users can now install the app.
