# Southville Mobile App - Installation Guide

## 📱 Installation Instructions

This app is **signed and safe** by Expo EAS (Expo Application Services). The security warnings you see are normal for apps installed directly (not from Google Play Store).

### ✅ Pre-Installation Checklist

- [ ] Android device running Android 5.0 (Lollipop) or higher
- [ ] At least 50 MB of free storage space
- [ ] Internet connection for first-time setup

---

## 🔐 Step-by-Step Installation

### Step 1: Enable Installation from Unknown Sources

1. Open **Settings** on your Android device
2. Navigate to **Security** (or **Apps** → **Special access** on newer devices)
3. Find **"Install unknown apps"** or **"Install from unknown sources"**
4. Select your browser or file manager (Chrome, Files, etc.)
5. Toggle **"Allow from this source"** to **ON**

**Note**: On Android 8.0+, you need to enable this per app (browser/file manager).

### Step 2: Download the APK

Choose one of these trusted distribution methods:

#### Option A: Google Drive (Recommended)
1. Open the Google Drive link provided
2. Tap the APK file
3. Tap **"Download"** or **"Open with"** → **"Package Installer"**

#### Option B: HTTPS Website
1. Visit the provided HTTPS website URL
2. Tap the download button/link
3. The download will start automatically

#### Option C: Firebase App Distribution
1. You'll receive an email invitation
2. Click the link in the email
3. Follow the on-screen instructions to install

### Step 3: Install the APK

1. Once downloaded, tap the notification or open **Files** app
2. Navigate to **Downloads** folder
3. Tap the APK file (e.g., `Southville-Mobile-1.0.0.apk`)
4. You may see a security warning - this is **normal and safe**
5. Tap **"Install anyway"** or **"Install"**
6. Wait for installation to complete (usually 10-30 seconds)

### Step 4: Launch the App

1. Tap **"Open"** after installation, or
2. Find **"Southville-Mobile"** in your app drawer
3. Tap to launch the app

## ⚠️ Understanding Security Warnings

### Why Do I See Warnings?

Android shows security warnings because:
- ✅ The app is **not from Google Play Store** (internal distribution)
- ✅ Android's **Play Protect** hasn't scanned it yet
- ✅ This is **normal behavior** for directly-installed apps

### Is This App Safe?

**Yes!** This app is:
- ✅ **Properly signed** by Expo EAS (Expo Application Services)
- ✅ **Built using secure practices** (HTTPS-only, minimal permissions)
- ✅ **Distributed by your school/organization** (trusted source)
- ✅ **No malware or harmful code** - verified during build process

### What Should I Do?

1. **Verify the source**: Make sure you downloaded from an official link
2. **Check the package name**: Should be `com.southville8b.mobile`
3. **Tap "Install anyway"**: The warning is expected for non-Play Store apps
4. **Trust the app**: After first launch, Android will remember your choice

## ❓ Troubleshooting

### "Install blocked" or "Can't install app"

**Solution**: 
- Go to Settings → Security → Install unknown apps
- Enable for your browser/file manager
- Try downloading again

### "App not installed" error

**Solutions**:
1. **Check storage space**: Free up at least 50 MB
2. **Uninstall old version**: If updating, uninstall first
3. **Download again**: File may be corrupted
4. **Check Android version**: Requires Android 5.0+

### "Parse error" or "Corrupted file"

**Solution**:
- Delete the downloaded APK
- Download again from the official source
- Ensure download completed fully

### App crashes on launch

**Solutions**:
1. **Restart your device**
2. **Clear app data**: Settings → Apps → Southville-Mobile → Storage → Clear Data
3. **Reinstall the app**
4. **Contact support** with error details
